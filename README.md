# 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?

## Answer : In javascript `map()` is basically Used to transform an array, Returns a new array, Does not modify original array. On the other hand `forEach()` used for Used to loop over array, Does NOT return anything.


# 3) What is the difference between `==` and `===`?

## Answer : In Javascript we use `==` for Compares values and Performs type coercion and `===` used for Compares value AND type and No type conversion


# 4) What is the significance of `async`/`await` in fetching API data?

## Answer : In javascript `async`/`await` is important cause it Makes async code look like synchronous, Easier to read, Better error handling with try/catch, Very important for real-world API development.


# 5) Explain the concept of Scope in JavaScript (Global, Function, Block).

## Answer : In javascript Scope controls variable accessibility 
## Global scope : Accessible everywhere.
## Function scope : Accessible only inside function.
## Block scope : Variables declared with let and const inside {}.
